Project: Real-Time Retail Anomaly Detection

Requirements:
Python 3.x

Install libraries:
pip install pandas numpy scikit-learn dash plotly joblib

Steps to Run:

1. Train the model
python train_model.py

2. Run the dashboard
python dashboard.py

3. Open browser:
http://127.0.0.1:8050

The dashboard will show live anomaly detection on transactions.
